# DESIGN PATTERNS

This project is for learning purposes. It contains examples of simple implementations for the most common design patterns in software development.
Contains examples for the following patterns:
* Creational Patterns
	- Factory
	- Abstract Factory

* Structural Patterns

* Behavioral Patterns


## Compile and Run

1. Go to any example directory and run:
	```bash
	javac *.java
	```
2. Run the compiled program: 
	```bash
	java Main
	```
